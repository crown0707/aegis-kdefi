🖥️📱 반응형 디자인

이 사이트는 PC, 태블릿, 스마트폰 모든 화면에 최적화되어 있습니다. 별도의 모바일 버전이 따로 있지 않고, 한 번 만든 페이지가 화면 크기에 맞춰 자동으로 모양을 바꿉니다.

  • 메뉴, 카드, 영상 모두 화면 크기에 맞춰 자동 정렬
  • 글자 크기도 모바일에서 읽기 좋게 자동 조정
  • 가로/세로 모드 전환에도 자연스럽게 대응

🏠 홈 화면에 추가하기 (앱처럼 사용)

한 번 설정해두면, 매번 브라우저를 열 필요 없이 홈 화면 아이콘 한 번 탭으로 바로 진입할 수 있어요.

📱 iPhone (Safari)
  1. 사이트 열기 → 하단의 공유 버튼 탭
  2. "홈 화면에 추가" 선택
  3. 이름 확인 후 "추가" 탭

→ 이제 앱처럼 홈 화면에서 바로 실행!

🤖 Android (Chrome)
  1. 사이트 열기 → 우측 상단 메뉴 (⋮) 탭
  2. "홈 화면에 추가" 선택
  3. "추가" 확인

→ 이제 앱처럼 사용 가능!

💬 카톡 1:1 문의 활용

학습 중 궁금한 점이 있으면 바로 물어볼 수 있도록, 카톡 문의 버튼을 모든 페이지에 띄워두었어요.

  • 모든 페이지 우측 하단에 항상 노출
  • 모바일에서는 탭 한 번이면 카톡 채팅방으로 바로 연결
  • 스크롤을 해도 따라다니기 때문에 언제든 호출 가능
